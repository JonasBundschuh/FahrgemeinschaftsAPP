# FahrgemeinschaftsAPP

## 🚌 FahrgemeinscaftsAPP is an app to create, manage carpools.

### Having the following features:


> 🔰 1. Driver Options \textcolor{red}{red}
- Add Drivers
- Display Drivers

> 🧍 2. Member Options
- Add Members
- Display Members
> ✨ 3. Carpool Options
- Add Carpools
- Display Carpools
- Join / Leave Carpools
> ⚙️ 4. Settings
 - Change Theme
 - Change Username
 - Change Password
